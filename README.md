# marocAffiliate Docs

This repository contains the project documentation website built with MkDocs.

## Quick start

1. Create and activate a virtual environment:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the docs locally:

```bash
mkdocs serve
```

Open `http://127.0.0.1:8000` in your browser.

## Build static site

```bash
mkdocs build
```

The generated static site is in `site/`.
